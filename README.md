## Publish with GitHub Pages

1. Sign in to GitHub and create a new repository named `tran-gia-bao-portfolio`.
2. Set the repository to **Public**.
3. Upload all files and folders from this package to the repository root.
4. Commit the files to the `main` branch.
5. Open **Settings → Pages**.
6. Under **Build and deployment**, choose **Deploy from a branch**.
7. Select branch **main** and folder **/(root)**, then click **Save**.
8. Your site will be available at:

```text
https://YOUR_GITHUB_USERNAME.github.io/tran-gia-bao-portfolio/
```

## Recommended finishing edits

- Replace the CSS initials avatar with a professional headshot if desired.
- Add project screenshots to an `assets/images/` folder and place them in project cards.
- Add a GitHub profile link after the final username is known.
- Review whether you want your phone number shown publicly before publishing.

## Local preview

Open `index.html` directly in a browser, or run a simple local server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.
